Examples

This page will show examples of how to use the application.

Inputting URLs

To input a URL for analysis, use the following endpoint:

    POST /input-url
    Content-Type: application/json

    {
      "url": "https://www.youtube.com/watch?v=example_video_id"
    }

Providing User Feedback

To provide user feedback, use the following endpoint:

    POST /user-feedback
    Content-Type: application/json

    {
      "feedback": {
        "comment_id": "example_comment_id",
        "corrected_sentiment": "positive"
      }
    }

Saving Data in CSV Format

To save data in CSV format, use the following endpoint:

    POST /save-csv
    Content-Type: application/json

    {
      "data": [
        {
          "comment": "example_comment",
          "vader_sentiment": {"neg": 0.1, "neu": 0.8, "pos": 0.1, "compound": 0.0},
          "hf_sentiment": {"label": "neutral", "score": 0.8}
        }
      ],
      "filename": "output.csv"
    }

Creating Heatmaps

To create a heatmap of sentiment data, use the following endpoint:

    GET /visualization/heatmap

Creating Word Clouds

To create a word cloud of comment data, use the following endpoint:

    GET /visualization/wordcloud

Creating Sentiment Distribution Charts

To create a sentiment distribution chart, use the following endpoint:

    GET /visualization/sentiment-distribution

Real-Time Sentiment Trends

To view real-time sentiment trends, use the following endpoint:

    GET /real-time

Filtering Data

To filter sentiment data based on specific criteria, use the following endpoint:

    POST /filter
    Content-Type: application/json

    {
      "criteria": {
        "date": "2023-01-01",
        "sentiment": "positive"
      }
    }
